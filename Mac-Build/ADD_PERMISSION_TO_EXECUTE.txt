If you try to open the Gradescope2Canvas.command file and get an error saying it doesn't have the permission to run, please do the following

1. Open Terminal
2. Type "cd " (no quotes, but DO include the space after cd)
3. Drag the folder containing the Gradescope2Canvas.command file into the Terminal window
    - Folder name CANNOT contain spaces
    - command should look something like cd /Users/username/Desktop/Gradescope2Canvas (possibly with different words between the / marks)
4. Press enter
5. Type "chmod +x Gradescope2Canvas.command" (no quotes)
6. Press enter
7. Try to open the Gradescope2Canvas.command file again, it should work now
